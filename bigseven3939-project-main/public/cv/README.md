# Thư mục CV

Upload file CV của bạn vào thư mục này với tên chính xác:

`NguyenTruongBao-CV.pdf`

Cách upload qua giao diện GitLab:
1. Vào **Repository > Files**, mở thư mục `public/cv/` (nhánh `add-cv-section` hoặc nhánh chính sau khi merge)
2. Bấm nút **+** > **Upload file**
3. Chọn file PDF, đặt tên `NguyenTruongBao-CV.pdf` và commit

Sau khi merge vào nhánh chính, GitLab Pages sẽ tự động deploy và CV sẽ hiển thị trên website.
